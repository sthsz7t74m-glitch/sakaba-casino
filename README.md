# 宴会ゲームBOX

1台のスマホを囲んで遊べる、ポップな宴会ゲーム集です。チップやBETなどのカジノ要素はありません。

## 収録ゲーム

- フォーダイスちんちろ（logic-arcade版 v1.0.7をそのまま収録）
- どすこい
- NGワード
- 究極の二択
- 爆弾しりとり
- ジェスチャー
- 5秒ぴったり
- 全員一致
- ワードウルフ
- 第一印象ゲーム
- 禁止文字トーク
- カタカナ禁止
- ランキング当て
- 以心伝心お絵描き
- 3秒で3つ
- 3ヒントクイズ
- 共通点さがし

## 開発

```bash
npm install
npm run dev
```

## 本番ビルド

```bash
npm run build
```

## GitHub Pages

```bash
npm run build:github
```

生成された `docs` フォルダを、GitHub Pagesの公開元（`main /docs`）に設定します。
