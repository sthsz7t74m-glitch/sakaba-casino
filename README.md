# 酒場カジノ

飲み会で1台のスマホを囲んで遊べる、酒場カジノ風の宴会ゲーム集です。

## 収録ゲーム

- ちんちろ
- どすこい
- ハイ＆ロー
- ラッキー7
- ルーレット
- コイントス

共通チップ機能を備え、スマートフォンの縦画面を中心に遊べるよう設計しています。

## 開発

```bash
npm install
npm run dev
```

## 本番ビルド

```bash
npm run build
```

## GitHub Pages

```bash
npm run build:github
```

生成された `docs` フォルダを、GitHub Pagesの公開元（`main /docs`）に設定します。
